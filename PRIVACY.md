# Privacy Boundary

This repository intentionally contains only the reusable content-engine code, setup files, and sanitized templates.

It does **not** include:

- Article drafts or published articles
- Hero images or generated media
- Private idea catalogues
- Publishing calendar/state
- QA reports, metrics, growth reports, or social copy
- API keys, tokens, credentials, or `.env`
- User-specific Obsidian vault content

Runtime data belongs outside git or in ignored local files.
